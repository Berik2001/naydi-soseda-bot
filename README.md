# 🏠 Roommate Bot — поиск сожителей в Казахстане

Telegram-бот для поиска сожителей: анкета, фильтры по городу/полу/бюджету и взаимные мэтчи (как в дейтинге, но для жилья).

**Стек:** Python 3.9+ · [aiogram 3.x](https://docs.aiogram.dev) · PostgreSQL (Supabase) · asyncpg · python-dotenv

---

## ✨ Возможности

- Последовательная анкета из 14 шагов через **FSM** (пол, цель, город, район, бюджет, привычки, фото, «о себе» и т.д.)
- Валидация бюджета (10 000–5 000 000 ₸) и текста «о себе» (≤200 символов)
- Карточка-превью перед сохранением: **Сохранить** / **Заполнить заново**
- Поиск кандидатов `/search` с фильтрами:
  1. тот же город;
  2. пол совпадает с предпочтением;
  3. разница в бюджете ≤30%;
  4. без показа самого себя;
  5. без повторов (таблица `views`).
- Лайк / супер-лайк / пропуск и уведомление обоим при **взаимном мэтче**
- Редактирование анкеты по полям, пауза/возобновление показа

## 💬 Команды

| Команда | Описание |
|---|---|
| `/start` | Начало / регистрация анкеты |
| `/profile` | Посмотреть свою анкету |
| `/edit` | Изменить анкету по полям |
| `/search` | Найти сожителей |
| `/pause` | Скрыть анкету из поиска |
| `/resume` | Снова показывать анкету |
| `/help` | Справка |

## 🚀 Запуск

### 1. Зависимости

```bash
python3 -m venv venv
source venv/bin/activate          # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### 2. Настройка окружения

```bash
cp .env.example .env
```

Заполни в `.env`:
- `BOT_TOKEN` — токен от [@BotFather](https://t.me/BotFather);
- `DATABASE_URL` — строка подключения к PostgreSQL.

### 3. База данных

Таблицы (`users`, `views`, `likes`) создаются **автоматически** при первом запуске.

> **Supabase (важно):** прямой хост `db.<ref>.supabase.co` работает только по IPv6.
> Если у провайдера нет IPv6 — используй строку **Session Pooler** (IPv4):
> `postgresql://postgres.<ref>:<password>@aws-1-<region>.pooler.supabase.com:5432/postgres`.
> Символ `@` в пароле кодируй как `%40`.

### 4. Старт

```bash
python bot.py
```

В логах должно появиться `Бот запущен. Ожидаю сообщения...`

## 🗂 Структура проекта

```
roommate_bot/
├── bot.py                 # точка входа: Dispatcher, роутеры, пул БД
├── texts.py               # все тексты и словари значений
├── .env.example           # шаблон переменных окружения
├── database/
│   ├── db.py              # пул asyncpg + CRUD и логика мэтчинга
│   └── models.py          # SQL-схема таблиц
├── handlers/
│   ├── start.py           # /start /profile /edit /pause /resume /help
│   ├── registration.py    # FSM-анкета (14 шагов)
│   └── matching.py        # /search, лайки, мэтчи
├── keyboards/
│   └── inline.py          # все inline-клавиатуры
├── states/
│   └── form.py            # состояния FSM
└── requirements.txt
```

## 🔒 Безопасность

- `.env` добавлен в `.gitignore` и **не** попадает в репозиторий.
- Если токен или пароль БД утекли — перевыпусти токен через `/revoke` у @BotFather и смени пароль БД в Supabase.
